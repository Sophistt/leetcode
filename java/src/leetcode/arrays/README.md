# Arrays 题解

## 题目列表

### [56. 合并区间](Solution_56.java)
**核心思路**: 按左端点排序，遍历时比较当前区间与结果集最后一个区间是否重叠，重叠则合并右端点，否则加入新区间。

### [189. 轮转数组](Solution_189.java)
**核心思路**: 三次反转法 - 整体反转，前 k 个反转，后 n-k 个反转。O(1) 空间复杂度。

### [238. 除自身以外数组的乘积](Solution_238.java)
**核心思路**: 构造左右乘积数组，`res[i] = L[i] * R[i]`，其中 L[i] 是左侧所有元素乘积，R[i] 是右侧所有元素乘积。

### [274. H 指数](Solution_274.java)
**核心思路**: 排序后从后向前遍历，当 `citations[i] > h` 时 h 递增，直到不满足条件。
